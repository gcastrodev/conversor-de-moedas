# 💱 Conversor de Moedas

Um projeto simples de **conversão de moedas** desenvolvido com **HTML, CSS e JavaScript**, que utiliza uma **API de câmbio em tempo real** para converter valores de **Real Brasileiro (BRL) para Dólar Americano (USD)**.

## 🚀 Funcionalidades

* Converter valores de **Real (R$) para Dólar (US$)**
* Consumo de **API externa de taxas de câmbio**
* Interface simples e intuitiva
* Formatação automática de moeda
* Layout responsivo básico

## 🛠️ Tecnologias utilizadas

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **ExchangeRate API**

## ⚙️ Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/conversor-moedas.git
```

2. Acesse a pasta do projeto:

```
cd conversor-moedas
```

3. Abra o arquivo `index.html` no navegador.

## 🌐 API utilizada

O projeto utiliza a API:

https://www.exchangerate-api.com/

Para obter as taxas de câmbio atualizadas.
